## 📝 About The Project
**Soaply** is a specialized e-commerce platform designed to manage and sell artisanal handmade products. It bridges the gap between traditional craftsmanship and modern web technology, providing a seamless shopping experience for customers and a powerful management tool for business owners.

Built with a **Decoupled Architecture** (Laravel API + React SPA), this project focuses on performance, scalability, and clean code principles.

---

## 🚀 Key Features

### 💻 Frontend (React.js)
*   **Dynamic Product Catalog:** Smooth filtering and real-time search functionality.
*   **State Management:** Optimized cart logic and user session handling (Context API/Axios).
*   **Responsive UI:** Mobile-first design ensures a perfect look on all devices.
*   **Smooth Navigation:** Handled via React Router for a true SPA experience.

### ⚙️ Backend (Laravel API)
*   **RESTful API Standards:** Clean, versioned, and well-structured endpoints.
*   **Secure Authentication:** Powered by **Laravel Sanctum** for safe user and admin sessions.
*   **Advanced Database Schema:** Optimized MySQL relationships for products, categories, and orders.
*   **Image Management:** Efficient handling for high-quality product images using Laravel storage.

---

## 🛠️ Tech Stack

*   **Backend:** PHP 8.x, Laravel 11 (API Mode).
*   **Frontend:** React.js, JavaScript (ES6+), Tailwind CSS / Material UI.
*   **Database:** MySQL (Relational Schema).
*   **API Client:** Axios with interceptors for Token management.
*   **Tools:** Git, GitHub, Postman, Composer, NPM.

## 🔧 Installation & Setup

To get a local copy up and running, follow these simple steps:

### 1. Clone the repository
```bash
git clone [https://github.com/Rami-Almasri/soap-store.git](https://github.com/Rami-Almasri/soap-store.git)
2. Backend Configuration (Laravel)
Bash
cd backend
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
php artisan serve
3. Frontend Configuration (React)
Bash
cd frontend
npm install
npm start
